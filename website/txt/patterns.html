Coil Correction to Ensure Shell Continuity

As a result of solving the integer Diophant equation, we obtain a table of variants of geodesic coil correction to ensure the continuity of the shell.
The geodesic coil is uniformly transformed into a non-geodesic coil with different degrees of deviation.
It is recommended to choose the option with the lowest percentage deviation.
The key parameter is the Convenience, which determines the coil pattern and the degree of interlacing of the layers.
For some (not for all) Convenience values there are a set of options with period correction within specified limits.
Choose the Convenience value and the coil correction option from the table that suits you.
In the diagram you can observe the order of tape stacking corresponding to the selected variant.
Using the buttons at the bottom of the screen you can set the mode of displaying a single coil, N+1 coils or continuous shell coverage, where N is Convenience.

Coil Correction

  • Select the desired Convenience Number of wedges in the winding pattern.
  • Hit the Mesh structure flag if you want to create a mesh structure.
  • Select the ribbon laying variant in the layer from the table, focusing on the minimum percentage of correction of the coil period, compared to the geodetic one.
  • Click the Correct coil button to correct the coil period.
  • Using the buttons at the bottom of the screen you can set the mode of displaying coils.
