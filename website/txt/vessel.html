What WinderCAM is?

WinderCAM is an online CNC program calculation service for winders, producing composite structures
The service allows you to calculate a CNC program for winding on a four-axis machine with conoidal roller output.
You can upload your Own mandrel profile in csv format with any r and x order,
or load sample projects from the Examples section to explore the capabilities of our service.

Applications

Service is suitable for a wide range of applications in composite engineering:

  • Pressure Vessels & Storage Tanks: Gas cylinders (COPV), pressurized domes, ellipsoids, isotensoid shapes, spherical and cylindrical tanks.
  • Aerospace & Mobility Components: Drive shafts, structural spars (including non-circular), lightweight beams, aerodynamic tubes.
  • Industrial Infrastructure: Pipes, lighting poles, tapered masts, pipe reducers, custom shafts.
  • Mesh & Shell Structures: Lattice frameworks, mesh structures, transition geometries, composite adapters.
  • Marine & Energy Applications: Buoyancy tanks, masts, pressure housings, wind turbine components, oil & gas elements.

Workflow

Workflow navigation is organized through the tabs on the left:

  • (start page): session management and examples
  • Mandrel: profile upload and preprocessing
  • Coil: Geodesic path calculation
  • Patterns: Path correction to ensure full surface coverage or config a mesh structure
  • Winding: Motion trajectory generation for the winding machine and export to CNC program
  • Thickness: Layer thickness computation

Select a tab on the left to view a detailed description of each step.

 Start page

On the start page you can manage your session:

  • With Save session button you can download the session on your device as a file
  • With Load session button you can restore the session back from file
  • With the help of Clear session button you begin from scratch
