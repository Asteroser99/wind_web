Instructions

  1. Select the file with profiles of the formers by pressing the Load button.
  2. Draw the shell by clicking the Shell button.
  3. Select the layer number.
  4. Select the initial values for the pole hole radius and ribbon width.
  5. Draw a geodesic coil by pressing the Coil button.
  6. Select the desired number of patterns.
  7. Get variants of ribbon laying in the layer by pressing the Pattern button.
  8. Select the ribbon laying variant in the layer, focusing on the minimum percentage of correction of the coil period, compared to the geodetic one.
  9. After selecting the laying variant, the laying pattern will be displayed.
  10. Use the Clear button to clear the image.
