On this page you can use some tools to prepare the mandrel profiles in advance.

A mandrel is described by a set of points whose coordinates r = r(x)

Column x must contain strictly increasing values of the profile abscissa.

Mandrel import

Import mandrel from your CSV fileI with by clicking Import mandrel button.

Coordinates should be listed in two columns. Column positions should be determined by the header row

Format example:

        x, r
        2.1, 1.2
        3.3, 1.8
        4.5, 2.3

Mandrel transformation

You can use basic transformations with mandrel using Scale and Transform actions.

Keep in mind that the mandrel curve is directional. The starting point of the curve is marked with a circle on the graph.

Handle this block with caution — improper processing may render the mandrel unsuitable for further use.

  • Shift: shift coordinate on shift value.
  • Multiply: scale coordinate on K.
  • Swap: swaps coordinates X vs ℝ.
  • Reverse: rotates the mandrel about X axis.
  • Redirect: changes the direction of the mandrel curve.
  • Mirror: Completes the reflection of the mandrel curve relative to its start. Be careful - the operation is irreversible!

Finaly we get a complete mandrel profile, ready for further calculations
