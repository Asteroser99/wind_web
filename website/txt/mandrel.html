Preparing your mandrels for calculation

✦ Our platform allows you to use some tools to prepare your mandrel profiles in advance.
✦ A mandrel is described by a set of points whose coordinates r = r(x) are listed in two columns,
and the position of column r can be specified in the dialog, so you can use any of your csv files for import.
✦ Import your file with a full or half profile of the mandrel and transform it to get a complete profile ready for further calculations.

Mandrel import

Import mandrel from CSV file by clicking Import mandrel button.

Format example:


        x, r
        3.4, 1.2
        2.3, 1.2
        4.5, 2.3

Mandrel transformation

You can use basic transformations with mandrel using Transform actions.

Keep in mind that the mandrel curve is directional. The starting point of the curve is marked with a circle on the graph.

Handle this block with caution — improper processing may render the mandrel unsuitable for further use.

  • Swap: swaps coordinates X vs ℝ.
  • Reverse: rotates the mandrel about X axis.
  • Redirect: changes the direction of the mandrel curve.
  • Mirror: Completes the reflection of the mandrel curve relative to its start. Be careful - the operation is irreversible!
