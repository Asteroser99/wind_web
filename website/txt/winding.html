This page allows you to visualize how the winding machine executes a CNC-controlled motion program based on your selected parameters.

Calculating the Winding Process

  • Select the desired Safety ℝ — the offset distance between the roller center and the mandrel surface.
  • Specify the required G-code length — the number of frames (steps) in the CNC program.
  • Click the Winding button to calculate and simulate the coordinate motion.

Download the CNC Program

Click the Download CNC button to save the generated program.

Make sure to verify the CNC program format to ensure compatibility with your specific CNC controller. The generated motion data from WinderCAM can be used for actual winding, as long as your machine’s kinematic diagram matches the simulation model.

Visualisation

Use the mouse to rotate the simulation scene

Winding visualisation

Use the buttons at the bottom of the screen to select how the winding is displayed:

  • First coil — animated view of the initial coil is shown.
  • First touch — displays N+1 coils, where N is the selected Convenience value.
  • Full winding — shows the complete shell coverage with the tape.

Visibility Controls

The visibility and transparency of individual elements can be adjusted using the control buttons located at the bottom right of the screen. Each button corresponds to a specific visual component:

  • Mandrel
  • Lines
  • Tape
  • Equidistanta

Animation Controls

Use the Video player controls to play, pause, and step through the winding animation.
